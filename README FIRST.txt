5) pmd-report_after.csv - PMD report μετά τις αλλαγές

6) Φάκελος CoverageHTML Περιέχει export report του code coverage σε μορφή html

7) Φάκελος onlineHotel περιέχει όλο το project (βάση, source code, libs κλπ)

	OnlineHotel->bin - Περιέχει τα binaries

	OnlineHotel->db  - Περιέχει τη βάση mysql που φτιάξαμε db.sql, οδηγίες εισαγωγής (DBsetup) και βοηθητικό script (user.mysql).

	OnlineHotel->lib - Περιέχει τη library του mysql connector που χρησιμοποιήσαμε

	OnlineHotel->src->onlinehotel->gui (κώδικας που υλοποιεί το gui)
	
	OnlineHotel->src->onlinehotel->model (κώδικας που υλοποιεί το μοντέλο)
	
	OnlineHotel->src->onlinehotel->users (κώδικας σχετικά με τους χρήστες και αρχείο με usernames/passwords)

	OnlineHotel->src->testcases - Περιέχει τα juinit tests
